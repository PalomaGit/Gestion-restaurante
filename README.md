# Gestor de Reservas — Restaurante

## ¿Qué es esto?

Este proyecto es una aplicación web sencilla para gestionar las reservas de un restaurante. Con ella puedes:

- Crear nuevas reservas (cliente, teléfono, mesa, fecha, hora, número de personas, notas).  
- Ver todas las reservas en una tabla.  
- Editar o eliminar reservas existentes.  
- Buscar reservas por nombre o teléfono.  
- Ordenar la lista de reservas por distintos campos (cliente, fecha, mesa…).  

Todo con una interfaz clara y amigable, usando **Tailwind CSS**.

## Lo que hace

- Añadir reservas con datos completos  
- Listar todas las reservas de forma ordenada  
- Buscar rápido por nombre o teléfono  
- Ordenar por fecha, cliente, mesa, etc.  
- Editar o eliminar reservas  
- Experiencia responsiva (se adapta a móvil y escritorio)  

---

## Requisitos

- PHP (recomendado versión 7.x o superior)  
- Base de datos MySQL o MariaDB  
- Extensión `mysqli` habilitada  
- Configurar conexión en `db.php`  
- Tailwind CSS (se carga vía CDN en este proyecto)
